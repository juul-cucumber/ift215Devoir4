Page d'aide du site Pro-Gramme.


Accueil Inscription Produits Points de ventes Connexion Panier Commande

Sur cette page vous trouverez tous les explications pour chaque page du site Pro-Gramme.

Accueil

La page d'accueil est l'introduction de notre site web. Vous pouvez retrouver une petite description de l'entreprise. De plus, la page d'accueil contient les informations comme notre siège social, notre adresse e-mail et notre numéro de téléphone si jamais vous voulez nous contacter personnellement.


Inscription

C'est sur la page d'inscription que vous pouvez créer un nouveau compte, afin de pouvoir passer une commande à la fin de votre magasinage. Vous y retrouverez les informations sur les différents types d’abonnement disponibles.


Produits

En haut de la page, vous retrouverez des filtres pour catégoriser les types de produits et une barre de recherche si vous cherchez un produit en particulier. Finalement en haut à droite de la page se trouve un bouton qui vous redirigera vers votre panier.


Points de ventes

C'est sur la page points de ventes que vous retrouverez l’adresse de toutes nos succursales ainsi que leurs heures d’ouvertures.


Connexion

La page connexion sert à vous connecter si vous avez déjà créé un compte. En vous connectant, vous pouvez personnaliser votre panier et récupérer les items que vous avez déjà mis dans votre panier lors de votre dernière visite.


Panier

La page panier contient tous les items que vous avez ajoutés dans votre panier. Cette page est seulement accessible si vous êtes connecté à votre compte. Sur cette page vous pouvez retirer des produits de votre panier. De plus, vous pouvez voir le montant total de votre panier.


Commande

La page commande est la place où vous finalisez votre transaction. Elle est seulement accessible à partir de la page panier. Vous devez entrer vos informations personnelles afin de procéder à la livraison, ainsi que les informations sur votre méthode de paiement.
